## Creating Your First Project

1. Login to your Link Software account
2. On the dashboard, click **"New Project"**
3. Enter a project name and optional description
4. Click **"Create"**
5. Your project is now ready to organize documentation

## Where to next?

Add members or start writing docs right away from your project page.
