## Upgrade Process

Follow these steps to upgrade to the latest version.

<Steps>
  <Step title="Backup" icon="database">

    Create a backup of your current docs.

````bash
cp -r /path/to/your/docs /path/to/backup
````

  </Step>
  <Step title="Download" icon="download">

    Fetch the latest release.

````bash
curl -L https://github.com/linksoftware/releases/download/v1.2.0/linksoftware-v1.2.0.tar.gz | tar -xz
````

  </Step>
  <Step title="Install" icon="package">

    Run the installer.

````bash
npm install linksoftware@latest
linksoftware upgrade
````

  </Step>
  <Step title="Verify" icon="check-circle">

    Test your site.

    Visit `https://dashboard.example.com` and confirm all pages load correctly.

  </Step>
</Steps>
