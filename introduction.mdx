## Quick Start

Get up and running in minutes with these essential steps.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://app.linksoftware.com` and create your free account. No credit card required.
  </Step>
  <Step title="Create a Project" icon="plus">
    Click "New Project" and name it, like "API Docs". Add team members via email invites.
  </Step>
  <Step title="Upload Docs" icon="upload">
    
````bash
# Example: Import from GitHub
git clone your-repo.git
link-software import ./docs --project="API Docs"
````
    
    Use the CLI or drag-and-drop interface for instant setup.
  </Step>
  <Step title="Publish" icon="rocket">
    Preview changes, then hit "Publish". Share the live URL with your team.
  </Step>
</Steps>

## Integrations and Extensibility

Embed Link Software anywhere with simple API calls.

<CodeGroup tabs="JavaScript,Python">
```javascript
// Fetch latest docs
const response = await fetch('https://api.example.com/v1/docs?project=API Docs', {
  headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
});
const docs = await response.json();
```
```python
import requests

response = requests.get(
    'https://api.example.com/v1/docs',
    params={'project': 'API Docs'},
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
)
docs = response.json()
```
</CodeGroup>
